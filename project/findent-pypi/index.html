findent PyPi

Oct 27, 2021
Code Project

What is it?

This is a wrapper around findent, the robust and performant Fortran formatting tool to permit easy installation with pip.

Installation

Normal installation using pip

pip install findent

Supported platforms

OS Version Architecture
Linux All with GLIBC 2.27+ x64
MacOS 10.15+ x64
Windows 10 and Server x64

What is it not?

This is not the repository for findent. Any bugs, questions or feature requests related to findent should be redirected to https://github.com/wvermin/findent/issues, the official findent GitHub repository or to SourceForge.
